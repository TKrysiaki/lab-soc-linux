# Portfólio SOC Analyst (Linux + SIEM + Detection Engineering)

Este repositório demonstra experiência prática em operações de segurança (SOC), incluindo:

- Detecção de ataques de força bruta (SSH)
- Análise de logs (auth.log, auditd)
- Correlação de eventos em SIEM (Wazuh)
- Investigação de incidentes
- Detecção e mitigação de ameaças

## 🖥️ Ambiente

- Kali Linux (Atacante)
- Ubuntu Server (Alvo)
- Wazuh (SIEM)
